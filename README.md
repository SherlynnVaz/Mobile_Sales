# Mobile Sales Analytics Dashboard (Power BI)

## Project Title

Mobile Sales Performance Dashboard using Power BI

---

## Description

This project presents an interactive Power BI dashboard designed to analyze mobile sales data across multiple dimensions such as city, brand, model, payment method, and time (day and month). The dashboard provides a comprehensive view of sales performance, customer behavior, and transaction trends.

---

## Purpose

The objective of this dashboard is to:

* Analyze sales trends and patterns
* Identify top-performing brands and mobile models
* Understand customer purchasing behavior
* Enable data-driven decision-making

---

## Tech Stack

* Power BI Desktop – Data visualization and dashboard creation
* Microsoft Excel (.xlsx) – Data source
* DAX (Data Analysis Expressions) – Measures and KPIs
* Power Query – Data cleaning and transformation

---

## Data Source

Dataset: Mobile Sales Data (Excel file)

The dataset includes:

* Sales amount
* Quantity sold
* Transaction details
* Customer ratings
* Payment methods
* City and date information

---

## Features and Highlights

### Key KPIs

* Total Sales: 769M
* Total Quantity: 19K
* Total Transactions: 4K
* Average Sales Value: 40K

---

### Sales Analysis

* Sales by City (map visualization for geographical insights)
* Sales by Mobile Model (comparison of top-performing devices)

---

### Time-Based Insights

* Sales by Day Name
* Quantity Trends by Day
* Monthly filtering for dynamic analysis

---

### Customer Insights

* Distribution of customer ratings (1 to 5 scale)
* Helps evaluate customer satisfaction

---

### Transaction Insights

* Payment method breakdown:

  * UPI
  * Debit Card
  * Credit Card
  * Cash

---

### Interactive Filters

* Month-wise filtering
* Brand selection
* Mobile model selection
